IndiGo Airlines Crisis — YouTube Comment Text Analytics
This project applies end-to-end text analytics to public discourse surrounding the IndiGo Airlines flight cancellation crisis, scraped from 63 YouTube videos (43,692 comments). Using Python-based NLP, the project covers the full pipeline from raw text preprocessing to advanced analytics — built as part of an MBA Text Analytics course and mapped to SDG 16: Peace, Justice and Strong Institutions.
What this project covers

Text preprocessing — tokenization, stemming, lemmatization, stopword removal
Bag of Words, TF-IDF, N-grams, Word2Vec embeddings
Sentiment classification using VADER, Naive Bayes, SVM and Logistic Regression
Aspect-based sentiment analysis across 8 crisis dimensions
Topic modelling using LDA and LSA
Document clustering using K-means and Hierarchical (Ward linkage)
Named Entity Recognition using rule-based and spaCy
Extractive and abstractive text summarization

Dataset
43,692 YouTube comments scraped from 63 videos covering the IndiGo cancellation crisis. Preprocessed and cleaned to 27,139 English comments across 28 feature columns.
Tools & Libraries
Python NLTK spaCy Gensim scikit-learn SciPy VADER Matplotlib Pandas
SDG Alignment
SDG 16 — Peace, Justice and Strong Institutions. The project uses NLP to quantify public demand for accountability from IndiGo and its regulator DGCA.
